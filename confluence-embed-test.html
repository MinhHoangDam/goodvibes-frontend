ðŸ“‹ Instructions for Confluence

Step 1: Start your dev server or use your hosted URL

npm start

Step 2: Open this HTML file to test the embed

Step 3: When ready to deploy to Confluence, use this HTML code:

<iframe 
  src="YOUR_HOSTED_URL_HERE" 
  width="100%" 
  height="700px" 
  frameborder="0"
  style="border: none; border-radius: 8px;"
></iframe>

Step 4: In Confluence, edit a page and:

  • Type /html or /iframe
  • Paste the iframe code above
  • Replace YOUR_HOSTED_URL_HERE with your actual hosted URL
  • Publish the page!
